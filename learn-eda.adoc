= Learn about NetApp Workload Factory for EDA
:icons: font
:imagesdir: ./media/

[.lead]

Workload Factory for EDA helps you optimize FSx for ONTAP usage performance, prevent capacity constraints, and automate storage provisioning by integrating it with Infrastructure as Code (IaC), ensuring consistent and reliable use by workloads.

Workload Factory for EDA includes a dashboard that provides a centralized view of your FSx for ONTAP file systems and volumes used for EDA workloads. 

It offers the following key benefits: 

* It streamlines the management of multiple FSx for ONTAP file systems for EDA workloads. 
* It offers centralized views, analysis, and tools to help you optimize storage operations, enhance performance, and maintain operational health across AWS regions and file systems. 
* It reduces job runtime and accelerates time to market.
